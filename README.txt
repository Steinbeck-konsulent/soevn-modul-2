Steinbeck – Søvn Modul 2
========================

Repository: soevn-modul-2

Filerne skal ligge direkte i repositoryets hovedmappe:

- index.html
- apptekst_modul2_v1.txt
- README.txt

GitHub Pages
------------
1. Gå til Settings → Pages.
2. Vælg “Deploy from a branch”.
3. Vælg branch: main.
4. Vælg mappe: /(root).
5. Tryk Save.

Den forventede adresse er:
https://steinbeck-konsulent.github.io/soevn-modul-2/

Om appen
---------
- Mobiloptimeret interaktivt micro-dosis-forløb.
- Svar gemmes lokalt i brugerens browser via localStorage.
- Der anvendes ingen database eller login.
- Brugeren må ikke skrive navne eller andre identificerende oplysninger.
- Besvarelserne kan samles og gemmes som PDF via browserens udskriftsfunktion.
- Udviklet af Steinbeck og AI-understøttet.
