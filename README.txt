Modulets fokus
--------------
Modulet undersøger, hvordan et lille barn, der er trygt ved én voksen,
også kan blive trygt ved de andre voksne på stuen.

Forløbet fører medarbejderen gennem:
1. Kort opsamling fra arbejdet med søvnen.
2. Flere mulige forklaringer.
3. Mindst én faglig kilde.
4. Observation over to-tre dage af barnets vej mellem de voksne.
5. Én lille prøvehandling inden for de nuværende grupper og vagtplan.

Kilder i modulet

- EVA: “Forskere – Sådan får I trygge børn”.
- EVA TEMA 21: “Tryg tilknytning”, især side 4 og 6.
- EVA Input: “Tryg tilknytning”.
- EVA: “En tryg start i vuggestue og dagpleje”, især side 8-10.

Teknik og data
--------------
- Mobiloptimeret interaktivt micro-dosis-forløb.
- Svar gemmes lokalt i brugerens browser via localStorage.
- Der anvendes ingen database og intet login.
- Denne version bruger nye lagringsnøgler (v3), så gamle testsvar ikke
  automatisk føres med over i den nye version.
- Knappen “Nulstil” vises på alle trin og sletter kun Modul 2-svarene
  i den aktuelle browser efter bekræftelse.
- Besvarelser kan samles og gemmes som PDF via browserens udskriftsfunktion.

Datasikkerhed
-------------
Brugeren må ikke skrive navne eller oplysninger, der kan identificere børn,
familier eller kolleger. Appen sender ikke besvarelser til en database.
